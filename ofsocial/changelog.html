Openfire Social Plugin Changelog

0.0.6 -- April 8, 2016

  • Added Office 365 sign-in using ADAL

0.0.5 -- March 12, 2016

  • Updated WordPress to 4.4.2 and Buddypress to 2.5.1
  • Openfire Social is now a child plugin to the Fastpath plugin. It now depends on fastpath.jar plugin to work.
  • Openfire bookmarks are created when BuddyPress groups are created. Public groups are set for all openfire users while private groups are set for the openfire group only
  • When BuddyPress hidden group are created, Fastpath workgroups are created and the groups are added to the workgroup request queues.

0.0.4 -- February 4, 2016

  • Fixed minimum Java support to Java 7 (which is equal to Openfire 4.0.1)

0.0.3 -- February 1, 2016

  • Build for Openfire 4.0.1

0.0.2 -- Jan 6th, 2016

  • Upgraded to WP 4.4.1

0.0.1 -- Dec 13th, 2015

  • Initial release.
